\# Habit Tracker CLI



Консольное приложение для мониторинга привычек и анализа прогресса выполнения задач.



\## Технические характеристики

\* \*\*Стек технологий:\*\* Python 3.13, JSON.

\* \*\*Соблюдение стандартов:\*\* Код отформатирован согласно PEP 8 с использованием Black.

\* \*\*Статический анализ:\*\* Проверка выполнена с помощью линтера Ruff.

\* \*\*Хранение данных:\*\* Файловая база данных в формате JSON.



\## Функциональные возможности

\* Создание новых записей о привычках.

\* Автоматический подсчет серии ежедневных выполнений (streak).

\* Визуализация текущего статуса всех активных привычек в консоли.

\* Удаление записей и обновление локального хранилища в реальном времени.



\## Инструкция по развертыванию

Для запуска приложения необходимо склонировать репозиторий и запустить основной скрипт:



1\. `git clone https://github.com/Goydaa/habit\_tracker.git`

2\. `cd habit\_tracker`

3\. `python main.py`



\## Контроль качества кода

В проекте настроены инструменты для обеспечения качества кодовой базы:

\* `black main.py` — автоматическое форматирование.

\* `ruff check main.py` — линтинг и поиск ошибок.
